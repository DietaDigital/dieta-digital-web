# Dieta Digital Web

Sitio oficial de Dieta Digital, creado con Next.js, TypeScript y Tailwind CSS.

## Primera etapa

- Landing profesional
- Pagina de precios
- Pagina de beta
- Politica de privacidad
- Terminos
- Contacto

Sin pagos, sin login y sin Supabase todavia.

## Desarrollo local

```bash
npm install
npm run dev
```

Abrir:

```text
http://localhost:3000
```

## Deploy

El proyecto esta preparado para deploy en Vercel conectando el repositorio de GitHub.
